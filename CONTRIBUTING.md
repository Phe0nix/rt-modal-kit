## Contributing to rt-modal-kit

Thanks for your interest in contributing! A few simple guidelines to get your changes accepted quickly:

- Fork the repository and create a feature branch from `develop`.
- Run the dev server: `npm install` then `npm run dev`.
- Follow the coding style (TypeScript, CSS Modules). Run `npm run lint` before submitting.
- Open a descriptive pull request with issue reference and rationale for the change.

If your change is non-trivial, open an issue first to discuss the approach.
