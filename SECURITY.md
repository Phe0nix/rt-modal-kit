# Security Policy

## Supported Versions

This project currently supports the latest published version on npm.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report vulnerabilities by emailing: security-report@proton.me

Include the following details so we can triage quickly:
- Package version
- Affected environment (OS, browser, React version)
- Reproduction steps or proof of concept
- Impact assessment

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Fix timeline: depends on severity and complexity

If the report is valid, we will publish a fix and include advisory details in release notes.
